myflow v3 — 5-Stage Pipeline

One clear path. Each stage produces an artifact consumed by the next. Cross-cutting practices run alongside.

1. Discover & Align

shaping
brainstorming discover explore
Typical flow: brainstorm → discover (formalize) → explore (if multiple paths)
Artifact: .rpiv/artifacts/discover/ or .rpiv/artifacts/solutions/
Checkpoint: capturing-learnings
▼ artifact handoff ▼

2. Research & Design

deciding
research design architecture-review blueprint plan
Two paths: design → architecture-review → plan (complex) or blueprint (fast, fused design+plan)
Artifact: .rpiv/artifacts/plans/ (canonical — implement consumes this)
Checkpoint: capturing-learnings
● epiphany-tabling active
▼ artifact handoff ▼

3. Implement

building
implement test-driven-development subagent-driven-development dispatching-parallel-agents verification-before-completion
Primary: implement executes the plan phase by phase
Inner loop: TDD · subagents · parallel dispatch (inside each phase)
Artifact: Working tree changes
● epiphany-tabling active; create-handoff / resume-handoff as needed
▼ artifact handoff ▼

4. Validate & Review

verifying
validate code-review receiving-code-review revise
Artifact: Validation report + Review (.rpiv/artifacts/reviews/)
Gate: code-review loops until zero blockers (max 3 passes). Covers quality, security, and dependencies.
Checkpoint: capturing-learnings
▼ artifact handoff ▼

5. Land & Learn

closing
group 1 · commit & document
commit as-built-documentation
group 2 · reflect & reconcile
writing-retros capturing-learnings
group 3 · update & close
AGENTS.md memory reconcile status review resolve tabled.md finishing-a-development-branch
Artifacts: docs/changes/ · docs/retros/ · docs/memory/ · docs/status.md · git commits

Cross-Cutting Practices

epiphany-tabling

Active during stages 2–4. Anything not trivially in-scope → docs/tabled.md. Resolve in stage 5 group 3.

capturing-learnings

Checkpoint after stages 1, 2, 4, 5. Promote twice-seen patterns to skills/memory/runbooks.

create / resume handoff

Available at any stage. Save state mid-pipeline, resume in fresh session.
